# Chat?
## A Terminal Chatbot using Swift Foundation Models

A simple command line chatbot that uses the [Swift Foundation Model](https://developer.apple.com/documentation/FoundationModels) in MacOS 26.

## Usage:

### Simply just ask chat

```console
$ chat, what are some cheap date ideas 
```

```console
$ chat, what are some recipes that include chicken that is about to go bad
```
